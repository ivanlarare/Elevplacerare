❓ Guide: Automatisk Elevplacerare 3000

Välkommen till ditt nya favoritverktyg för bordsplaceringar och grupper!

1. Lägg in din klass

I rutan Klasslista till vänster skriver du in namnen på alla elever. Skriv ett namn per rad. (Programmet ignorerar stora/små bokstäver).

2. Rita ditt klassrum (För bordsplacering)

Klicka på ⚙️ Inställningar (Layout). "Rita" upp bänkarna:

  • X = En bänk
  • - (bindestreck) = Ett tomrum / en gång

3. Sätt upp regler (Anpassningar)

Inne i inställningarna klickar du på ❌ Öppna bredvid Anpassningar.

  • Längst fram/bak: Gäller enbart för bordsplaceringen i klassrummet.
  • MÅSTE sitta ihop: Skriv in elever som ska jobba bredvid varandra eller hamna i samma grupp. Skriv ett par per rad, separera med kommatecken.
  • Får INTE sitta ihop: Skriv in de par som behöver arbetsro från varandra.

4. Skapa placering eller grupper

Bordsplacering: Klicka på den gröna knappen Placera i Klassrummet! för att fylla dina ritade bänkar.

Grupper: Fyll i hur många grupper du vill ha i rutan längst ner till vänster, och klicka därefter på den lila knappen Slumpa Grupper!. Programmet skapar då jämna grupper som fortfarande följer alla dina regler om vilka som får och inte får vara ihop.

5. Spara och Skriv ut

  • Spara / Ladda: Klicka på 💾 Spara för att ladda ned en fil med din klass till datorn. Nästa gång klickar du på 📂 Ladda för att få tillbaka allt direkt.
  • Skapa PDF: Klicka på den orangea knappen 📄 Spara som PDF / Skriv ut. Välj "Spara som PDF" i utskriftsrutan för att spara filen!
