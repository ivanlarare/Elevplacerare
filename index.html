❓ Guide: Automatisk Elevplacerare 3000

Välkommen till ditt nya favoritverktyg för bordsplaceringar! Programmet pusslar automatiskt ihop din klass baserat på hur ditt klassrum ser ut och vilka regler du sätter upp.

1. Lägg in din klass

I rutan Klasslista till vänster skriver du in namnen på alla elever. Skriv ett namn per rad. (Programmet ignorerar stora/små bokstäver).

2. Rita ditt klassrum

Klicka på den blå knappen ⚙️ Inställningar (Layout). Här "ritar" du upp hur bänkarna står i rummet med hjälp av två tecken:

  • X = En bänk
  • - (bindestreck) = Ett tomrum / en gång

Exempel för parbänkar med en mittgång:
X X - X X

3. Sätt upp regler (Anpassningar)

Inne i inställningarna klickar du på knappen ❌ Öppna bredvid Anpassningar. Här kan du bestämma fyra viktiga saker:

  • Längst fram: Skriv namnen på de som måste sitta på första raden (t.ex. Anna, Kalle).
  • Längst bak: Skriv namnen på de som ska sitta längst bak.
  • MÅSTE sitta ihop: Skriv in elever som ska jobba bredvid varandra. De kommer hamna horisontellt bredvid varandra utan någon gång emellan. Skriv ett par per rad, separera med kommatecken.
  • Får INTE sitta ihop: Skriv in de par som behöver arbetsro från varandra.

4. Slumpa placeringen!

Stäng inställningarna och klicka på den stora gröna knappen Placera Elever!. Programmet testar nu tiotusentals kombinationer tills det hittar en placering som följer alla dina regler.

5. Spara och Skriv ut

  • Spara / Ladda: Klicka på 💾 Spara för att ladda ned en fil med din klass till datorn. Nästa gång klickar du på 📂 Ladda för att få tillbaka allt direkt.
  • Skapa PDF: Klicka på den orangea knappen 📄 Spara som PDF / Skriv ut. Välj "Spara som PDF" i utskriftsrutan för att spara filen!
