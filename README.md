🤖 Guide: Automatisk Elevplacerare 3000
Välkommen till ditt nya favoritverktyg för bordsplaceringar! Programmet pusslar automatiskt ihop din klass baserat på hur ditt klassrum ser ut och vilka regler du sätter upp.

1. Lägg in din klass
I rutan Klasslista till vänster skriver du in namnen på alla elever. Skriv ett namn per rad. (Programmet bryr sig inte om stora eller små bokstäver, så Anna och anna räknas som samma elev).

2. Rita ditt klassrum
Klicka på den blå knappen ⚙️ Inställningar (Layout). Här "ritar" du upp hur bänkarna står i rummet med hjälp av två tecken:

X = En bänk

- (bindestreck) = Ett tomrum / en gång

Exempel för parbänkar med en mittgång:
X X - X X
X X - X X

3. Sätt upp regler (Anpassningar)
Inne i inställningarna klickar du på knappen ❌ Öppna bredvid Anpassningar. Här kan du bestämma tre viktiga saker:

Längst fram: Skriv namnen på de som måste sitta på första raden (separera med kommatecken, t.ex. Anna, Kalle).

Längst bak: Skriv namnen på de som ska sitta längst bak.

Får INTE sitta ihop: Skriv in de elever som behöver arbetsro från varandra. Skriv ett par per rad, med ett kommatecken emellan (t.ex. Vimes, Nobby).

MÅSTE sitta ihop: Skriv in elever som ska jobba bredvid varandra. De hamnar alltid axel mot axel utan någon gång emellan. Skriv ett par per rad och separera med kommatecken (t.ex. Nanny, Granny).

4. Slumpa placeringen!
Stäng inställningarna och klicka på den stora gröna knappen Placera Elever!. Programmet testar nu tusentals kombinationer tills det hittar en placering som följer alla dina regler. Blev det inte perfekt? Klicka bara på knappen igen för en ny variant!

5. Spara och Skriv ut
Spara / Ladda: Längst upp till vänster finns knappar för att spara din klass. När du klickar på 💾 Spara laddas en liten fil ned till din dator. Nästa vecka kan du klicka på 📂 Ladda, välja filen, och få tillbaka exakt samma klass, klassrum och regler igen.

Skapa PDF: Klicka på den orangea knappen 📄 Spara som PDF / Skriv ut längst ner. Välj "Spara som PDF" i utskriftsrutan, eller skriv ut den direkt på papper för att sätta upp på tavlan!
